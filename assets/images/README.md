# Images

Drop your real files in here, using these exact names (or update the paths in
index.markdown / projects.markdown if you'd rather rename them):

- assets/images/profile.jpg          -> your headshot, shown on the homepage
- assets/images/projects/project1.jpg -> PCB/hardware project photo #1
- assets/images/projects/project1-schematic.jpg -> optional schematic image
- assets/images/projects/project2.jpg -> project photo #2
- assets/images/projects/project3.jpg -> project photo #3

Recommended: JPG/PNG, roughly 800x600px or larger, under ~500KB each so the
site stays fast on GitHub Pages.
